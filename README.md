ClinicaMed API
Descrição do Sistema

A ClinicaMed é uma API REST desenvolvida em ASP.NET Web API para gerenciamento de consultas em uma clínica médica.
O sistema permite cadastrar, consultar, atualizar e remover agendamentos, utilizando Entity Framework Core para persistência dos dados em banco.
